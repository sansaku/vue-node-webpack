## Requirements

## Install
```shell
yarn
```

## Usage
```shell
yarn run dev # development
yarn run build # prod
```

## Includes

1. [Vuejs 2](https://vuejs.org/) (router / store / resouce / filter / directive ...)
2. Webpack 2 (ESLint / ES6 / Stylus / HMR / Proxy / Extract / Minify ...)
